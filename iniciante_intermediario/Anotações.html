Anotações :


  • Comando < HTML > é utilizado para abrir o site , fechando-o com < /HTML >;

  • Comando < HEAD > é utilizado para abrir o cabeçário da pagina, fechando-o com < /HEAD >;

  • Comando < TITLE > é utilizado para abrir o título da pagina, lá nas abas abertas, fechando-o com < /TITLE >;

  • Comando < BODY> é utilizado para abrir o corpo da pagina, fechando-o com < /BODY >; Podemos colocar dentro do comando BODY :

    1. < TEXT="COR" > é utilizado para mudar a cor do texto da página, como um todo, fechando-o com;

    2. < BGCOLOR="COR" > é utilizado para mudar a cor do fundo da página, fechando-o com;

  • Comando < /BR > é utilizado para pular uma linha, dando o famoso enter;

  • Comando < HR > é utilizado para criação de uma linha horizontal; Podemos colocar dentro do comando HR :

    1. < WIDHT="X%" > para mudar a porcentagem horizontal que ela ocupa;

    2. < SIZE="X" > para mudar a altura da linha ( verticalmente );

    3. < COLOR="COR" > é utilizado para mudar a cor da linha horizontal;

  • Comando < B > é utilizado para destacar algo em negrito, fechando-o com < /B >;

  • Comando < I > é utilizado para destacar algo em itálico, fechando-o com < /I >;

  • Comando < U > é utilizado para destacar algo, sublinhando, fechando-o com < /U >;

  • Comando < S > é utilizado para destacar algo, tachando, fechando-o com < /S >;

  • Comando < H > é utilizado para aumentar ou diminuir o tamanho da letra, colocando um valor X ao lado direito do H, sendo este de 1 ( maior ) à 6
    ( menor ), fechando-o com < H"X" >;

  • Comando < FONT > não é utilizado sozinho, fechando-o com < /FONT >, porém temos os seguintes atributos :

    1. < COLOR="COR" > é utilizado para mudar a cor de um texto específico; ou < COLOR="CÓDIGO DA COR >

    2. < FACE > é utilizado para definir o nome da fonte do texto;

    3. < SIZE="X" > ou < SIZE="+X" > ou < SIZE="-X" > utilizados para mudar o tamanho da fonte;

  • Comando < P ALIGN="LEFT/CENTER/RIGHT/JUSTIFY" > é utilizado para mudar a formatação do texto, para à esquerda, centralizado, à direita ou justificado, respectivamente; Fechando-o com < /P >

  • Comando < OL > é utilizado para ordenação com números à esquerda do texto ( como utilizado nos atributos do código font ), fechando-o com
    < /OL >;

  • Comando < UL > é utilizado para ordenação desordenada ( com um pontinho, por exemplo ), fechando-o com < UL >;

    1. Para o OL temos o atributo < TYPE="X" > colocando qualquer letra, com o intuito de indicar o tipo da numeração ( letras/algarismos minúsculos ou maiúsculos e números )

    2. Para o UL temos o atributo < TYPE="FORMA" > colocando uma forma, como por exemplo : square, circle, triangle, disk e etc;

    3. E, para ambos, utilizamos o comando < LI > para criar uma nova linha, fechando-o com < /LI >;

  • Comando < DL > para criar uma lista detalhada, com título e conteúdo, fechando-o com < /DL >;

    1. Temos o < DT > para titular o item da lista, fechando-o com < /DT >;

    2. Temos o < DD > para descrever o item da lista, fechando-o com < /DD >;

  • Comando < IMG SRC="local onde a imagem está"/"nome da imagem"."formato da imagem" > para adicionar uma imagem;Com os seguintes atributos :

    1. < BORDER="X" > para criar uma borda de expessura x;

    2. < HEIGHT="X" > para mudar a altura da imagem;

    3. < WIDTH="X" > para mudar a largura da imagem;

  • Comando < MARQUEE > para criar uma animação no texto, rolando para cima, baixo ou diagonal. Fechando com < /MARQUEE >. Com os atributos :;

    1. < BEHAVIOR > Define como o texto é rolado dentro da área do letreiro. Os valores possíveis são scroll, slide e alternate. Se nenhum valor for especificado, o valor padrão será scroll;

    2. < BGCOLOR > Define a cor do plano de fundo do letreiro através do nome da cor (ex: red, blue) ou de um valor hexadecimal;

    3. < DIRECTION > Define a direção da rolagem do texto dentro do letreiro, os possíveis valores são: left, right, up & down. Se nenhum valor for especificado, o valor padrão será "left";

    4. < HEIGHT > Define a altura do letreiro em pixeis ou em um valor percentual;

    5. < HSPACE > Aplica a margem horizontal;

    6. < LOOP > Define o número de repetições da animação do letreiro. Se nenhum valor for especificado assumirá o valor padrão de -1, que significa que a animação será repetida infinitamente;

    7. < SCROLLAMOUNT > Define o intervalo de tempo entre cada animação de rolagem em milissegundos. O valor padrão é 85. Qualquer valor menor que 60 será ignorado e o valor 60 será usado, a menos que seja especificado como truespeed;

    8. < TRUESPEED > Por padrão, valores abaixo de 60 milissegundos são ignorados, a menos que o valor truespeed esteja presente, caso esteja estes valores são aceitos;

    9. < VSPACE > Aplica uma margem vertical em pixeis ou em valor percentual;

    10. < WIDTH > Define a largura em pixeis ou em um valor percentual;

  • < TABLE > Cria uma tabela, podendo ser inteiramente modificada, fechando-o com < /TABLE >, com os seguintes atributos;

    1. < TR > Para criar uma linha na tabela;

    2. < TD > Para criar uma cédula na linha; Podemos usar < TD COLSPAN="X" > para aplicar o número de colunas que uma cédula da tabela terá, x podendo assumir valores de 1 a 6;

    3. < ALIGN="LEFT/CENTER/RIGHT" > Para alinhar a tabela;

    4. < BGCOLOR="COR/CÓDIGO DA COR" > Para mudar a cor de fundo e do conteúdo da tabela;

    5. < BORDER COLOR="COR"> Para a cor da borda ; < BORDER WIDTH > Para a largura da borda ; < BORDER STYLE > Para o estilo da borda [ex : dotted (pontilhada), inset (metade da borda), dashed solid ( laterais normal com em cima e em baixo pontilhado) e etc];

    6. < CELLPADDING > Este atributo define o espaço entre o conteúdo de uma célula e a borda, exibida ou não, dela;

    7. < CELLSPACING > Este atributo define o tamanho, em porcentagem ou em pixels, do espaço entre duas células (tanto horizontal quanto verticalmente), entre o topo da tabela e as células da primeira linha, a esquerda da tabela e a primeira coluna, o direita da tabela e a última coluna e na parte inferior da tabela e a última linha;

    8. < FRAME > Define qual lado do quadro ao redor da tabela deve ser exibido. Pode ter os seguintes valores: above, below, vsides, hsides, lhs, rhs, border, box e void;

    9. < RULES > Este atributo enumerado define onde as regras, ou seja, as linhas, devem aparecer em uma tabela. Podendo assumir : none, groups, rows, columns e all;

    10. Estes são os principais, porém lembramos que existem mais alguns atributos mais específicos ( como por exemplo < SUMMARY >, para pessoas com deficiência audiovisuais);
